
[블록체인]
- 강의 이론에 부합한 단일 블록체인(분산X)을 제작함
- 이전 해시값과 서로 연결된 블록들을 생성해 단일 블록체인을 실습해봄
- 새로운 해시는 index, proof, prev_hash로 제작
- 정해진 연산에 따라 proof를 넣고 while문으로 +1을 하면서 나올때까지(선행제로가 4개나올때까지) 돌리고 최종적으로
쌓인 proof를 채굴된(새로운) 블록의 proof로 넣는다

1. 블록체인을 만들고 플라스크로 배포한뒤 로컬에서 포스트맨으로 확인해본다.
1.1) 블록을 만드는 API -> /mine_block
1.2) 체인을 확인하는 API -> /get_chain
1.3) 유효한 체인인지 확인하는 API -> /is_valid
